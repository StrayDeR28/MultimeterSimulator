# MultimeterSimulator

Небольшое задание на реализацию функционала мультиметра с архитекторй под паттерн MVC.

## Режимы работы мультиметра:
- Нейтральный, показывает 0;
- Сопротивление [Ω];
- Сила тока [А];
- Напряжение постоянного тока [V];
- Напряжение переменного тока [V~].
